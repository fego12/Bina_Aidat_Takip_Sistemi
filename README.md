# Aidat Takip Sistemi (Dues Tracking System)

Bu sistem, site/apartman yöneticilerinin aidat takibini kolaylaştırmak için geliştirilmiş bir web uygulamasıdır.

## 🚀 Özellikler

- 💰 Aidat takibi ve yönetimi
- 👥 Üye yönetimi
- 📊 Gelir-gider takibi
- 📈 Aylık raporlama
- 💳 Ödeme durumu takibi
- 📱 Mobil uyumlu tasarım

## 🛠️ Teknolojiler

- PHP 7.4+
- MySQL 5.7+
- Bootstrap 5
- HTML5
- CSS3
- JavaScript
